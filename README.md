## 📊 Project Title: Madhav Sales Dashboard

### 🔹 Objective
To analyze ecommerce sales data and generate actionable business insights.

### 🔹 Tools Used
- Power BI
- Excel

### 🔹 Key Insights
- Maharashtra generated highest profit
- Clothing category contributed most sales
- COD was most used payment mode

### 🔹 Features
- Interactive dashboard with slicers
- State-wise and category-wise analysis
- KPI tracking (Sales, Profit, Quantity)

### 🔹 Conclusion
This dashboard helps businesses make data-driven decisions.
